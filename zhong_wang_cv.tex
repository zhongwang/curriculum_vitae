% ============================ PROFILE ========================================
\section{Profile}
Computational biologist and research group lead specializing in \textbf{genome foundation models},
large-scale metagenomics, and scalable machine learning for the life sciences. Creator of
\textbf{GenomeOcean}, an efficient Mixture-of-Experts genome foundation model, and architect of
open-source genomic AI tooling adopted across the research community. Twenty-plus years bridging
high-performance computing, deep learning, and microbial genomics, with a sustained record of
mentorship, multi-institutional collaboration, and competitive resource acquisition.

% ============================ RESEARCH HIGHLIGHTS ============================
\section{Selected Research Highlights}
\begin{cvlist}
  \item \textbf{GenomeOcean.} Designed a microbial genome foundation model using an 8-expert Mixture-of-Experts (MoE) architecture; sparse-upcycling a dense model delivered a \textbf{1.87$\times$ inference throughput gain} and higher biological plausibility while using \textbf{2.5$\times$ fewer active parameters}. \emph{(Major revision in progress, Nature Methods.)}
  \item \textbf{Routing-aware probing.} Showed that MoE routing vectors capture functional biological specialization (e.g.\ tRNA / rRNA classification) without explicit supervision --- a new tool for unsupervised functional annotation.
  \item \textbf{Genome-Factory.} Built and maintains an open-source library unifying genomic foundation-model workflows (data prep, model loading, benchmarking); \textbf{accepted to ICML 2026}, 130+ GitHub stars.
  \item \textbf{genomeocean-sentinel (GOS) v1.0.} AI-detection framework for synthetic DNA signatures; LBNL software disclosure \#2026-126 (under review for open-source release).
\end{cvlist}
